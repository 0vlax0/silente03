## Periodo 2008-2021

### Sitio Web
 * __2008 nacimiento__. El primer sitio web se hospedó en la [comuna digital Tuxfamily](https://tuxfamily.org) y se construyó con [SPIP](https://spip.net) y base de datos MySQL. Mostraba un mapa interactivo con geoposicionamiento de los archivos fonográficos personales. En la WayBack Machine se pueden consultar [capturas de 2012 a 2018](https://web.archive.org/web/*/http://silente.tuxfamily.org/).
 * __2020 reconstrucción__. Actualización de SPIP y rediseño visual. Actualmente (2021) disponible en modo consulta [http://silente.tuxfamily.org](http://silente.tuxfamily.org)
